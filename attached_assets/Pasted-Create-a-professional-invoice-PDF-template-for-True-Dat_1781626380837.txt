Create a professional invoice PDF template for True Data Broadband Services Pvt. Ltd.

IMPORTANT:

Do not modify any invoice business logic, numbering logic, payment logic, snapshot logic, or database schemas.

Implement ONLY the PDF layout and invoice preview UI.

Use:

* ReportLab for PDF generation
* Existing invoice snapshot fields only

Never fetch live data from:

* company_settings
* customers
* subscriptions
* plans

Always use invoice snapshot fields.

==================================================
GENERAL REQUIREMENTS
====================

Page Size:

* A4 Portrait

Margins:

* 20mm on all sides

Theme Colors:

Primary: #1F4959
Primary Dark: #011425
Secondary: #5C7C89
Border: #D9E1E5
Text: #242424

Typography:

* Clean
* Professional
* Print-friendly
* Readable in grayscale

Currency:

Indian Rupee (INR)

Always use:

₹

Use Indian number formatting.

Examples:

₹799.00

₹12,499.00

₹1,25,000.00

Date format:

DD MMM YYYY

Example:

15 Jan 2025

Requirements:

* Support multi-page PDFs
* Keep table headers repeated across pages
* Add page numbers
* Prevent content overlap
* Ensure proper spacing throughout

======================

* Invoice is locked
* Payment is recorded

5. Charges table is easy to read
6. Currency uses ₹ symbol
7. Indian number formatting is used
8. Amount in words is displayed
9. Outstanding amount is prominent
10. PDF matches invoice preview
11. Page numbers work
